# 🏭 LOOMGAZER PANEL

A modern React-based Textile Factory Management Dashboard designed to monitor factory operations, track raw materials, manage machines, optimize production workflows, and improve industrial efficiency through Data Structures and Algorithms.

---

## 📌 Project Overview

LoomGazer Panel is a smart textile factory monitoring system developed using React JS. The dashboard helps factory operators manage materials, monitor machine performance, verify equipment, optimize power consumption, track production logs, and plan efficient processing routes.

The project combines real-world textile industry operations with various Data Structures and Algorithms concepts to provide a practical factory automation solution.

---

## 🎯 Problem Statement

Build a dashboard to monitor a textile factory, showing the status of raw materials, tracking machine settings, and managing the entire production line efficiently.

---

## 🚀 Key Features

### 1. Material Quality List
- Displays all textile material batches.
- Shows quality, GSM, and approval status.
- Includes material search functionality.

### 2. Machine Monitoring System
- Monitors machine efficiency and power usage.
- Allows dynamic machine status updates.
- Supports Running, Maintenance, and Offline states.

### 3. Production Log Queue
- Maintains production logs in First-In-First-Out order.
- Simulates factory production scheduling.

### 4. Equipment ID Checker
- Verifies equipment IDs against factory records.
- Displays machine details instantly.

### 5. Machine Setting Undo System
- Records machine setting modifications.
- Allows operators to undo recent changes safely.

### 6. Market Value Sorter
- Sorts fabric batches according to market value.
- Helps identify high-value production batches.

### 7. Factory Map Hub
- Visualizes machine connections and production flow.
- Represents factory layout using graph concepts.

### 8. Quickest Processing Planner
- Calculates the most efficient production route.
- Minimizes total processing time.

### 9. Power Load Balancer
- Analyzes machine power consumption.
- Recommends optimized load distribution.

---

## 🧠 Data Structures & Algorithms Used

| Feature | DSA Concept |
|----------|------------|
| Material Search | Array + Linear Search |
| Machine Monitoring | Array Traversal |
| Production Log Queue | Queue (FIFO) |
| Machine Setting Undo | Stack (LIFO) |
| Equipment ID Checker | Linear Search |
| Market Value Sorter | Bubble Sort |
| Factory Map Hub | Graph |
| Processing Planner | Greedy Approach |
| Power Load Balancer | Array Processing |
